# Ressie

## About

This project allow you to archive your playback of [Spotify](https://spotify.com) (using [IFTTT](https://ifttt.com)) and [Plex](https://plex.tv) (using [Tautulli](https://tautulli.com/)) to a Twtiter account using their API.
